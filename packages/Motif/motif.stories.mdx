## Examples
Sizes used here might not be exact. 
Use whatever Size is appropriate for your app.<br />
Define a class and apply it to your motif:
```css
.icon {
  width: 30px;
  height: 30px;
}
```
```html
<motif class="icon" type="api" />
```
