# @tillo-ds/text-input

## Install

```bash
npm install @tillo-ds/text-input
```

## Usage

In your Vue file:

```html
<template>
  <tillo-text-input/>
</template>
<script>
  import TilloTextInput from '@tillo-ds/text-input'

  export default {
    components: {
      TilloTextInput
    }
  }
</script>
```
