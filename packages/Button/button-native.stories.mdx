# Native Button

A button can be built up from the native browser button.<br/>
The class applied can reference a mixin to apply the desired style.

## Props
No props needed here as this is not a component, but an example <br/>
of how the button styles can be applied to a native button

```css
@import '@tillo-ds/button/styles';

.your-button-class {
  @include largeBtn($variant);
  // or
  @include mediumBtn($variant);
  // or
  @include smallBtn($variant);
}
```
```html
<button class="your-button-class"></button>
<!-- or -->
<a class="your-button-class" href="https://tillo.io" ></a>
```

## Stories

### Destrcutive styles
```scss
.destructive-large {
  @include largeBtn('destructive');
}
.destructive-medium {
  @include mediumBtn('destructive');
}
.destructive-small {
  @include smallBtn('destructive');
}
```

### As an outline
To complment the solid state, an outline style is available too.

```scss

  @include largeLineBtn('destructive');
}
.neutral-medium-outline {
  @include mediumLineBtn('neutral');
}
.positive-small-outline {
  @include smallLineBtn('positive');
}
```
