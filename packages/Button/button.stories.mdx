# Native Button

A button can be built up from the native browser button.<br/>
The class applied can reference a mixin to apply the desired style.

```css
@import '@tillo-ds/button/styles';

.your-button-class {
  @include largeBtn($variant);
  // or
  @include mediumBtn($variant);
  // or
  @include smallBtn($variant);
}
```


## Destrcutive styles
```html
<button class="destructive-[large|medium|small]"></button>
```
