# @tillo-ds/button

## Install

```bash
npm install @tillo-ds/button
```

## Usage

In your Vue file:

```html
<template>
  <tillo-button></tillo-button>
</template>
<script>
  import TilloButton from '@tillo-ds/button'

  export default {
    components: {
      TilloButton
    }
  }
</script>
```
